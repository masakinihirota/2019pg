#データ
list
双方向リスト。要素の挿入、削除が多い時に。

set
集合

map
連想配列

stack
スタックそのものをシミュレートするような問題に使います。

queue
幅優先探索などをするときに利用することが多いです。

priority_queue
貪欲なアルゴリズムを実装する場合などに使います。

pair
二つの値を一組にします。
// 各要素は first, second でアクセス可能

string
文字列型
isalpha(char);
isdigit(char);
islower(char);
isupper(char);
tolower(char);
toupper(char);
